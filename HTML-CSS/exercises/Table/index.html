Striped Effect:

For create a Striped effect on a table on or other things, we need to use the pseudoclass: "nth-child()"

  • "((amount))" set only one column
  • "((amount)n)" set a order for how much coluns get striped
  • "(odd)/2n-1" will stripe columns in a odd order
  • "(even)/2n" will stripe columns in a even order

Example:
