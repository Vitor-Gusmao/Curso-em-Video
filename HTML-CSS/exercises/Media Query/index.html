What are Media Queries in CSS

Media query is nothing more than a CSS structure that allows you to apply specific stylizations to a web page according to certain conditions, thus ensuring that this page can adapt to the screen layout in different sizes and types of media.

(you can use media query command inside of a style link)

Media Types:

  • (media="print") modify printers;
  • (media="screen") modify main screen
  • (media="handheld") modify handheld devices;
  • (media="all") modify all media types;

Media Features

  • "( media="# and (orientation: portrait) )"
  • "( media="# and (orientation: landscape )";
  • "( media="# and (min-width/height: # )"
  • "( media="# and (max-width/height: # )"
