Paragraph

We can create paragraphs in texts using text-ident.

Example:

Lorem ipsum dolor sit, amet consectetur adipisicing elit. Quos veritatis dicta eum nihil ad debitis illo culpa id unde beatae consequatur vero vitae quis aliquid, aperiam perspiciatis molestias reprehenderit deserunt.

This was a single <p>, so if u want the next phrases with paragraph you always will need to put the command in the next phrases. But if you put a <div> before and write the command, the nexts phrases will always come with paragraphs.

Example:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Provident nobis modi sed corporis porro in quos. Consequatur animi dignissimos repudiandae repellat, velit facilis, rerum reiciendis nihil temporibus mollitia sit error?
