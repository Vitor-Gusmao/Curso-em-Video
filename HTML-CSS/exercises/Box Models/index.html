Units:

We can change the size of the things that are comming in this article using the units:

  • px - The height of the element will be measured in pixels.
  • em - The height of the element will be a multiple of the font size of the element.
  • % - The height of the element will be a percentage of the height of its parent element.
  • vw - The height of the element will be a percentage of the width of the viewport.
  • vh - The height of the element will be a percentage of the height of the viewport.
  • rem - The height of the element will be a multiple of the font size of the root element.

Border

We can border a box using the css commands:

  • "border-width" for set the width of the border
  • "border-style" for set the border styles, they are solid, dashed,dotted,double,groove,ridge,inset and outset,
  • "border-color" for change the color of the border
  • "border-radius" for change the structure of the form, like instead of the border be a square, you can make it a circle or a triangle.
  • "padding(top, left, right and bottom.)" generate a space around an element's content, inside of any defined borders.
  • "margin(top, left, right and bottom.)" The margin property sets the margins for an element.
  • "outline-style" set a style for the line outside the borders. the styles are: none, hidden, dotted, dashed, solid, double, groove, ridge, inset, outset, initial, inherit.
  • "outline-width" set the size of the line outside the border.
  • "outline-color" set the color of the line outside the border.
  • "outline-offset" property adds space between the outline and the edge or border of an element. The space between an element and its outline is transparent.
